# Sobre o projeto

O PlayNuvem é um reprodutor de mídia que permite assistir a filmes e séries diretamente do seu armazenamento em nuvem pessoal. Trata-se de um projeto de estudo, atualmente em fase de desenvolvimento.

O PlayNuvem exibi as mídias com informações da biblioteca TMDB, como seções de recomendados, lançamentos e principais gêneros, e como fonte de mídias reprodutora conecta a parti do armazenamento em nuvem pessoal do Google Drive e reproduz direto no PlayNuvem.
